About OpenSky Network

The OpenSky Network is a non-profit association based in Switzerland that provides open access to real-world air traffic control data. This application uses the OpenSky Network API to display live flight information from aircraft around the world.

Each flight is tracked using its unique ICAO24 transponder address, and we display comprehensive information including position, altitude, velocity, and more.
